## Twinlife Framework Library

This is the Java library providing the core functionalities used by [twinme](https://www.twin.me), [twinme+](https://www.twin.me) and [Skred](https://www.skred.mobi):

- device registration to the signaling server,
- twincode management for the representation of peer relations,
- database storage (with SQLCipher) for local representation of relations, groups and storage of local conversations,
- WebRTC peer connection management with the signaling server
